# Video Chat Client

Простой React клиент для видеочата.

## Установка

```bash
cd client
npm install
```

## Запуск

```bash
npm run dev
```

Приложение откроется на `http://localhost:5173`

## Переменные окружения

Создайте файл `.env` в папке `client`:

```
VITE_WS_URL=ws://localhost:3000/ws
```

По умолчанию используется `ws://localhost:3000/ws`

## Использование

1. Откройте приложение в браузере
2. Введите ID комнаты (например, "room-1")
3. Нажмите "Присоединиться"
4. Разрешите доступ к камере и микрофону
5. Откройте ту же комнату в другом окне/браузере для тестирования

## Сборка для продакшена

```bash
npm run build
```

Собранные файлы будут в папке `dist/`
